# recall
An AI powered chatbot that remembers and visualises your chats as memories.

This is recall.
Your personal memory vault.

It uses a Gemini API Key for now, but will be changed in the future.

recall stores your most valuable memories and thoughts as bubbles on the recall sphere, where you can visualise your memories and interact with them. You can pick up a bubble and continue a conversation with the chat anytime.

recall respects your privacy, all data is stored on-device.
